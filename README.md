# ReactJs + Hooks + Redux - Building an Trello like app from scratch

## Requirements
* **nodejs >= v14.17.2**
* **npm >= v6.14.13**
* **yarn >= v1.22.11**


Clone project and run test.
```
$ git clone https://github.com/vinhqngo5/Mern-Trello-Client.git
$ cd mern-trello-client
$ nvm use 14.17.2 
$ yarn install
$ yarn start
```
